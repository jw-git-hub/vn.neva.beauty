### Типографика
- Заголовки: **Cormorant** (600), Google Fonts, кириллица.
- Текст/интерфейс: **Manrope** (400/500/600), кириллица.
- Подключение через Google Fonts с `preconnect` и `display=swap`.
- Иконки: SVG (Lucide). Эмодзи в роли иконок запрещены.

### Тон
Светлый, воздушный, fashion-минимализм. Тёплые градиенты, мягкие скругления,
много воздуха. Анимации 150–300 мс, уважают `prefers-reduced-motion`.

## Шаблон страницы услуги

Хедер → интро-герой с фото и кратким описанием → «Преимущества процедуры»
(4 карточки) → **прайс-лист** → блок записи → «Дополните ваш уход» (2–3 связанные
услуги) → футер.

### Прайс-лист (компонент)
- Табы по разделам (напр. Женщины / Мужчины), где они есть в исходных данных.
- КОМБО-программы — выделенными карточками сверху.
- Остальные позиции — строки «название … цена» с табличными цифрами (`đ`).
- На узких экранах разделы могут сворачиваться в аккордеон.
- Цены берутся дословно из исходной страницы; рендер статический (в HTML),
  чтобы работать без JS и индексироваться.

## Герой с ДНК-спиралью

Переписанный под ваниль `woven-light-hero` (исходник — Three.js + framer-motion):
- Геометрия — **двойная спираль ДНК** (две нити частиц + перемычки), а не torus knot.
- Частицы в розово-бронзовой гамме палитры, светлый градиентный фон.
- Медленное вращение + «дыхание» + мягкая реакция на курсор.
- framer-motion-анимации текста заменяются на CSS/Web Animations.
- `prefers-reduced-motion`: спираль статична либо заменяется статичным изображением.
- Three.js подключается ES-модулем **только** на главной.

## Техническая раскладка

```
vn.neva.beauty/                 # корень публикуемого репозитория
  index.html
  <slug>/index.html             # 10 страниц услуг + privacy/
  assets/
    css/  tokens.css base.css components.css home.css service.css
    js/   nav.js hero-helix.js price-tabs.js booking.js
    img/  (скачанные локальные изображения)
    icons/ (SVG Lucide)
  CNAME                         # vn.neva.beauty
  robots.txt  sitemap.xml  404.html  favicon
  .gitignore                    # .env, .superpowers/, ключи
scripts/
  fetch-images.(sh|py)          # скачивание Unsplash, читает ключи из .env (вне репо)
```

- Один файл — одна ответственность (CSS-партиалы по назначению).
- Без инлайн-стилей и инлайн-скриптов.
- Изображения: `loading="lazy"` ниже первого экрана, заданы width/height/aspect-ratio.

## Доступность и качество

- Контраст текста ≥ 4.5:1, проверить пары на палитре.
- Тач-цели ≥ 44px, видимые focus-состояния.
- Семантические заголовки h1→h6, alt у изображений, aria-label у иконочных кнопок.
- Мобайл-first, брейкпоинты 375 / 768 / 1024 / 1440.
- `prefers-reduced-motion` уважается во всех анимациях.

## Вне scope (YAGNI)

- Бэкенд, формы с отправкой на сервер, личный кабинет, онлайн-оплата.
- Мультиязычность (только русский).
- Секции «О центре» и «Галерея».
