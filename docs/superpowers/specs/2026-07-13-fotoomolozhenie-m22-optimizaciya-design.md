# Оптимизация страницы «Фотоомоложение M22» — дизайн

Дата: 2026-07-13
Проект: vn.neva.beauty (standalone статика, генератор Python + Jinja2)
Страница: `/fotoomolozhenie-m22/`

## Цель

Применить к странице услуги «Фотоомоложение M22» оба промта оптимизации
(`promt-1-seo.md` + `promt-2-ai-geo.md`) — следующая строка `[ ]` в
`TODO-optimization.md`. Девятая услуга после «Лазерной эпиляции» (довела
шаблон услуг до структурного уровня), «LPG-массажа», «Эндосфера-терапии»,
«Ультразвуковой кавитации», «SMAS-лифтинга», «Удаления татуировок»,
«Удаления перманентного макияжа» и «Morpheus 8». Работа **чисто контентная**,
в одном файле `content.yml`.

## Ключевой факт: структурный слой уже готов

`service.html.j2`, `build.py`, `schema.py` сами рендерят: крошки с учётом
вложенности, узел `Service` + `AggregateOffer` (числа из `prices.json`),
блок FAQ (guard `{% if svc.faq %}`, первый `<details>` открыт), подстановку
мессенджеров `render_faq_contacts`. Правок шаблонов/Python **не требуется** —
только `services.fotoomolozhenie-m22` в `content.yml`.

## Жёсткие ограничения (из промтов)

- **Не менять цены** — диапазон в schema считается автоматически из прайса.
- **Не выдумывать факты.** Реальные факты M22 со страницы/прайса:
  метод — **фототерапия световыми импульсами** на мультимодульной платформе
  M22; выравнивает тон, убирает возрастную пигментацию и сосудистые
  несовершенства, **работает с акне** (в прайсе строка «Лицо — Лечение акне»),
  стимулирует выработку **коллагена и эластина**; **встроенная система
  охлаждения** делает процедуру комфортной; подходит любому типу кожи
  (широкий набор фильтров). Зоны из `prices.json` — **лицо, шея, декольте,
  кисти рук**, плюс комбо (лицо+шея, шея+декольте) и отдельная программа
  лечения акне. **Курсовых пакетов (5/10) нет** — прайс по зонам и комбо.
- **Не менять URL** `/fotoomolozhenie-m22/`.
- Пользовательский контент — на русском; технические ключи — английские.

## Решения заказчика (подтверждено в брейншторме)

1. **Число сеансов — нейтрально, как Morpheus 8.** «Число процедур подбирают
   индивидуально на консультации» — **без слова «курс»**. В прайсе только
   разовые зоны, курсовых пакетов и чисел на сайте нет; консистентно с
   осторожной формулировкой Morpheus 8.
2. **Название метода — оставить как на сайте.** «Фототерапия / световые
   импульсы / платформа M22» — **без аббревиатуры IPL**. Не вводим новый
   термин, полностью консистентно с текущей формулировкой страницы и
   категорией «Лифтинг и омоложение» («M22 работает светом»).

## Честность формулировок

- «Больно ли» — фототерапия светом, **не** игольчатая/механическая; встроенная
  система охлаждения снижает тепловые ощущения, процедура комфортна, не
  повреждает кожу, без длительного восстановления. Entity-consistent с
  категорией «Лифтинг и омоложение» (там M22 «проходит комфортно и не
  повреждает кожу», в отличие от микроигольчатого Morpheus 8).
- «Сколько процедур» — число индивидуально **на консультации**, эффект
  нарастает по мере обновления кожи и выработки коллагена; без выдуманных
  чисел и без слова «курс».
- Никаких `Review`/`AggregateRating` (реальных отзывов нет).

## Что меняется — `content.yml → services.fotoomolozhenie-m22`

Два поля переписаны + новый ключ `faq`. `benefits`/`related`/`hero_image`/
`title`/`category`/шаблоны — не трогаем.

- **seo_title** ~42→~59 симв.: `Фотоомоложение M22 в Дананге — ровный тон
  кожи, Neva Beauty` (ключ в начале, город + бренд, диапазон 50–60).
- **seo_desc** ~102→~150 симв.: фотоомоложение на платформе M22 + Neva Beauty
  (Дананг) + свет выравнивает тон / пигментация / сосудистые несовершенства /
  акне + «По записи».
- **intro** (answer-first; идёт и в hero, и в `Service.description` как чистый
  текст без тегов): определение услуги + бренд + город + метод (фототерапия,
  световые импульсы платформы M22) + что решает (тон, возрастная пигментация,
  сосудистые несовершенства, акне, коллаген/эластин) + система охлаждения =
  комфорт + зоны (лицо/шея/декольте/кисти рук) + число процедур на
  консультации + «Приём по записи».
- **faq** — 5 реальных Q&A (стиль предыдущих услуг), первый `open`:
  1. Больно ли делать фотоомоложение M22? — фототерапия светом, не игольчатая/
     механическая; встроенная система охлаждения; не повреждает кожу; без
     длительного восстановления; интенсивность индивидуально.
  2. Сколько нужно процедур фотоомоложения M22? — число **индивидуально на
     консультации**; эффект нарастает по мере обновления кожи (коллаген);
     конкретное число заранее не фиксируем; без слова «курс».
  3. С чем работает фотоомоложение M22? — тон, возрастная пигментация,
     сосудистые несовершенства, акне, коллаген/эластин; зоны лицо/шея/декольте/
     кисти рук. Ссылки на `/smas-lifting/` и `/morpheus-8-rf-lifting/`.
  4. Сколько стоит фотоомоложение M22 в Дананге? — зависит от зоны и их
     сочетания (лицо, шея, декольте, кисти рук, комбо, отдельная программа
     лечения акне); «Цены» на странице.
  5. Как записаться? — WhatsApp/Telegram/Instagram (плейсхолдеры), приём по
     записи, ru+en.
